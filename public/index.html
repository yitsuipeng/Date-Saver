西區漫步
龍山寺 / 臺北華西街夜市 / 新富町文化市場 / 剝皮寮歷史街區 / 三味食堂 / 西門紅樓 / 西門町徒步區

很喜歡台北西區復古的氛圍，龍山寺旁邊的街區處處是故事。剝皮寮跟新富町文化市場，可以看到上個世紀的繁華市中心的影子，值得好好逛逛。附近的三味食堂也是遠近馳名的台式日菜店，鮭魚壽司必吃。如果喜歡順道逛逛街的人，旁邊就是西門町，走路坐車都很快就到，十分之方便。

飛機迷的浪漫
飛機巷 / 上引水產 / 林安泰古厝 / 臺北市立美術館 / MAJI MAJI集食行樂 / 雙城街夜市

飛機迷都知道飛機巷這個必朝聖景點吧!!! 看一個上午都不會膩。旁邊的上引水產可以滿足午餐，順便看看進口的大螃蟹有多可愛。下午可以到花博公園走走，離林安泰古厝跟北美館非常近，喜歡文藝的夥伴可以逛逛。圓山站旁邊的市集超適合假日拜訪一下，有農產品市集或是不定期的表演，非常熱鬧。這個地點附近也有很多夜市，今天是選了雙城街夜市來結束這一天

陽明山小旅行
夢幻湖 / 陽明山冷水坑 / 擎天崗大草原 / 文化大學後山情人坡 / 草山夜未眠 / 天母生活市集

本來是想去看看夢幻湖有多夢幻，結果還好，不過離冷水坑跟擎天崗都很近，就順道去看看牛牛，是很好的散步行程。 傍晚決定去吃個浪漫的晚餐，就坐公車到文化大學附近，到一間非常有名的夜景餐廳草山夜未眠。想說離天母蠻近的，吃完就一路漫步到山下，剛好碰上廣場的二手市集；不過山路有點暗，步行的人要小心喔。
